# Data Analysis Pipeline

## Overview
This project analyzes a CSV dataset and generates summary statistics, insights, and a formatted text report.

## Features
- Loads data from a CSV file
- Stores records as dictionaries
- Filters records by column value
- Calculates minimum, maximum, and average values
- Summarizes total records, unique genres, and average rating
- Generates written insights
- Exports a formatted report

## Dataset
This version uses `movies.csv`, which includes movie titles, years, genres, ratings, and values.

## How to Run
```bash
python data_analyzer.py

The program prints a summary in the terminal and creates a report.txt file.
